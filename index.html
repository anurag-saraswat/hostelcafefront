Why Learn Python?

Python is a general-purpose language. It is an interpreted, object-oriented, high-level programming language with dynamic semantics and simple easy-to-use syntax. It has wide range of applications from Web development (like: Django and Bottle), scientific and mathematical computing (Orange, SymPy, NumPy) to desktop graphical user Interfaces (Pygame, Panda3D).

Simple Elegant Syntax

Programming in Python is fun. It's easier to understand and write Python code. Why? The syntax feels natural. Even if you have never programmed before, its very easy to understand as well as write a code.

Not overly strict

You don't need to define the type of a variable in Python. Also, it's not necessary to add semicolon at the end of the statement. Python enforces you to follow good practices (like proper indentation). These small things can make learning much easier for beginners.

Increased Oppurtunities

Python has become a skill that is more in demand than ever, especially it can be integrated into web applications. Thus can be used to build desktop apps and web apps alike along with great support for data mining. Python is also the future of AI and Machine Learning.

The Future

Python is the 4th most popular programming language out of 100. Interest in learning Python grew by 22.1% in 2015 and has been to the 1st place when compared to other languages. Clearly, Python will continue to reign in terms of relevance, and has a pretty good future thanks to its large community.
